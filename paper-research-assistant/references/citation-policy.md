# Citation and evidence policy

## Evidence levels

- **Direct evidence:** The retrieved passage explicitly states the claim. Cite the title and PDF page.
- **Supported inference:** Several retrieved facts reasonably imply the claim. Label it as an inference and cite all supporting passages.
- **Unsupported:** The library does not provide enough evidence. Say so and do not manufacture support.

## Required checks

1. Read the full retrieved passage, not only the matching sentence.
2. Check whether the statement concerns assumptions, simulations, theory, or empirical findings.
3. Preserve qualifiers such as "under regularity conditions", "asymptotically", and "in simulations".
4. Do not generalize beyond the studied model, sample, estimator, or parameter regime.
5. Use `[Title, PDF p. N]`; add a short quotation only when wording itself matters.
6. Treat physical PDF page numbers and printed page numbers as different systems.

Never infer bibliographic metadata from memory when it is missing from the indexed source.
