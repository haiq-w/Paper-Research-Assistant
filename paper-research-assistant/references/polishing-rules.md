# Academic polishing rules

Default to moderate editing: improve grammar, precision, cohesion, and academic register while preserving meaning.

## Preserve exactly unless the user authorizes substantive revision

- Equations, variable names, mathematical operators, and parameter restrictions.
- Numerical results, units, table and figure references.
- Citation keys and reference markers.
- Whether a statement expresses association, prediction, identification, or causation.
- Strength qualifiers such as may, suggest, establish, and prove.

## Output

1. Provide the revised passage.
2. Explain only material changes.
3. Flag ambiguous source wording rather than guessing.
4. List claims requiring citations.
5. If library evidence is requested, search and cite it under the citation policy.

Do not add novelty claims, causal claims, or broader generalizations for rhetorical effect.
